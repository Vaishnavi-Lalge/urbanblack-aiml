# LongTripOpt вЂ” UrbanBlack ML Pricing Module

End-to-end ML pipeline for **fare prediction** and **rider acceptance (WTA)** for long-distance rides on the Urban Black platform (Pune region).

---

## Fare Rules (Locked)

| Component | Rule |
|---|---|
| Base fare | в‚№ 55 for first **1.5 km** |
| Short / medium (`< 15 km`) | в‚№ 25 / km (incremental after 1.5 km) |
| Long trip Tier-1 (`15 вЂ“ < 18 km`) | в‚№ 23 / km |
| Long trip Tier-2 (`18 вЂ“ < 20 km`) | в‚№ 22 / km |
| Long trip Tier-3 (`в‰Ґ 20 km`) | в‚№ 20 / km |
| Waiting | First **5 min free** в†’ в‚№ 2 / min |
| Night surcharge | **+ 25 %** on distance fare (22:00 вЂ“ 05:59) |
| Weather surge | **+ 10вЂ“20 %** on distance fare |
| Premium (AC) | **+ 10 %** on distance fare |
| GST | **5 %** on full sub-total |

**Shift constraints (enforced in dispatch, not fare calc):**
- Minimum 25 rides per 12-hour shift
- Minimum 135 km per shift
- Shift, depot, and centre-point fixed per allocation

## Workflow

### Step 1 вЂ” Generate synthetic training data
```bash
python src/generate_dataset.py
# в†’ data/synthetic_rides_dataset_50k.csv  (50 000 rows)
```

### Step 2 вЂ” Train models
```bash
python src/train.py
# Options:
#   --synthetic-csv  data/synthetic_rides_dataset_50k.csv
#   --real-csv       data/rides_dataset_clean.csv
#   --models-dir     models/
#   --real-oversample 50    # oversample real data Г—50 for weight
```

Expected metrics (on real held-out test):
- Fare model: RВІ в‰€ 0.96, MAPE в‰€ 6 %
- WTA model: AUC в‰€ 0.55, F1 в‰€ 0.96

### Step 3 вЂ” Single-trip inference
```bash
python src/predict.py --km 20 --vehicle premium --hour 23 --trip-type airport --wait 8
```

### Step 4 вЂ” Batch inference on CSV
```bash
python src/predict.py \
  --input-csv  data/rides_dataset_clean.csv \
  --output-csv data/predictions.csv
```

Output adds columns: `pred_fare_ml`, `pred_fare_rule`, `pred_fare_diff_pct`, `pred_wta_prob`, `pred_wta`, `pred_fare_slab`.

## Long-Trip Rate Reference

| Total km | Rate (incremental) | Slab ID |
|---|---|---|
| 0 вЂ“ 14.99 | в‚№ 25 / km | 1 |
| 15 вЂ“ 17.99 | в‚№ 23 / km | 2 |
| 18 вЂ“ 19.99 | в‚№ 22 / km | 3 |
| в‰Ґ 20 | в‚№ 20 / km | 4 |

> **Note:** The rate applies to the full incremental distance (beyond 1.5 km), not just the portion in each band. A 20 km trip pays в‚№20/km on all 18.5 km of incremental distance вЂ” not a blend.
